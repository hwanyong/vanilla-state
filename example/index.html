Vanilla State

Lightweight State Management for Vanilla JavaScript

English 한국어

What is Vanilla State?

Vanilla State is a lightweight state management solution for modern web applications. It provides proxy-based reactivity to maintain clean code while enabling effective state management.

Lightweight & Fast

Designed with zero dependencies, resulting in a very small bundle size.

Proxy-based Reactivity

Utilizes JavaScript Proxies for efficient state detection and update systems.

Framework Agnostic

Can be used with any framework including React, Vue, Angular, or plain JavaScript.

Installation

npm install @uhd_kr/vanilla-state

or

pnpm add @uhd_kr/vanilla-state

Usage

// ESM
import VnlState from '@uhd_kr/vanilla-state';

// Create a basic state
const count = new VnlState(0);

// Add event listener
count.addEventListener('change', (value) => {
  console.log('Count changed to:', value);
});

// Update state
count.set(count + 1);
