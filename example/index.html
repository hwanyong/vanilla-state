Vanilla State

가벼운 바닐라 자바스크립트 상태 관리 라이브러리

Vanilla State란?

Vanilla State는 모던 웹 애플리케이션을 위한 가벼운 상태 관리 솔루션입니다. Proxy 기반의 반응성을 제공하여 코드를 간결하게 유지하면서도 효과적인 상태 관리가 가능합니다.

가볍고 빠른 속도

외부 의존성 없이 설계되어 번들 사이즈가 매우 작습니다.

Proxy 기반 반응성

JavaScript Proxy를 활용한 효율적인 상태 감지와 업데이트 시스템을 제공합니다.

프레임워크 독립적

React, Vue, Angular 등 어떤 프레임워크와도 함께 사용할 수 있습니다.

설치 방법

npm install @uhd_kr/vanilla-state

또는

pnpm add @uhd_kr/vanilla-state

사용 방법

// ESM
import VnlState from '@uhd_kr/vanilla-state';

// 기본 상태 생성
const count = new VnlState(0);

// 이벤트 리스너 추가
count.addEventListener('change', (value) => {
  console.log('Count changed to:', value);
});

// 상태 업데이트
count.set(count + 1);
