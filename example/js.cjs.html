CommonJS와 ES Module 비교

      
CommonJS는 주로 Node.js 환경에서 사용되며, ES Module은 최신 브라우저와 Node.js에서 사용됩니다.


      
CommonJS 특징:

      
    
        
  • 동기적으로 모듈 로드
    • 
        
  • require()와 module.exports 사용
    • 
        
  • Node.js에서 기본적으로 지원
    • 
      


      
ES Module 특징:

      
    
        
  • 비동기 모듈 로드 지원
    • 
        
  • import와 export 키워드 사용
    • 
        
  • 정적 분석이 가능하여 트리 쉐이킹 최적화 지원
    •
